# Ashfrontier 美術禁用詞清單

## 摘要

本清單是 `Scripts/art_prompt_lint.py` 的人工可讀版本。它用來避免 prompt、metadata 與使用者可見內容混入外部作品專有名詞或直接風格模仿。

## 禁用詞

- `Kenshi`
- `Kenshi 2`
- `Lo-Fi Games`
- `Hiver`
- `Shek`
- `Holy Nation`
- `United Cities`
- `Old Empire`
- `Beak Thing`
- `Skin Bandit`
- `Skeleton`

## 備註

`Skeleton` 只允許在程式或技術文件中作為一般骨架概念使用，不允許在美術 prompt 中作為角色種族、造型、派系或視覺描述。
