# Ashfrontier v0.2 原創風格聖經

## 摘要

Ashfrontier v0.2 的美術目標是讓既有 playable prototype 脫離純灰盒，形成一致、可讀、可擴充的原創邊境荒漠視覺。這不是最終量產美術，而是用生成圖片、PBR 風格貼圖、UI icon、徽章、旗幟、portrait、場景 signage 與 decal 建立第一版視覺身份。

## 世界觀視覺

Ashfrontier 的世界是鹽風侵蝕的邊境盆地。城鎮不是乾淨的文明中心，而是被沙塵、鹽霜、缺水與舊工業殘件慢慢壓垮的殖民聚落。建築與道具要呈現「還能用，所以一直補」的狀態：金屬板重疊、布料曬白、砂岩牆剝落、木材焦黑、標誌被風沙磨損。

核心氛圍：

- 乾燥，不潮濕。
- 低科技，不高科技。
- 修補，不嶄新。
- 可玩原型，不追求不可維護的高複雜度。
- 原創世界，不模仿外部作品的專有造型。

## 派系辨識

### 鹽脊守望

- 定位：軍事、秩序、城門與配給。
- 形狀：直線、矛尖、階梯式護板。
- 色塊：灰白、暗鐵、低飽和藍灰。
- 用途：城門標記、禁區告示、守衛巡邏區旗幟。

### 玻璃屋商盟

- 定位：商貿、修補、工坊與倉儲。
- 形狀：圓弧、秤盤、鉚釘、貨牌。
- 色塊：氧化銅、褪色赭黃、舊布。
- 用途：商店招牌、交易告示、工坊標誌。

### 塵路行者

- 定位：荒野、游牧、拾荒與路線記憶。
- 形狀：斜線、裂片、繩結、陶片。
- 色塊：風沙褐、焦黑木、日曬紅布。
- 用途：破損方向牌、水源警告、荒野危險標記。

## UI 規則

- Icon 以物品功能可讀性優先，背景保持簡單。
- 64x64 預覽仍要能辨識主要剪影。
- 不使用外部品牌符號、真實國徽、現代貨幣符號或醫療標誌。
- 物品 icon 可使用低飽和 rim light，但不要做成發光戰利品。
- 派系 icon 必須可用在地圖、角色面板與通知，不依賴長文字。

## 場景規則

- 主城入口優先套用砂岩牆、城門標記、守衛旗幟與禁區告示。
- 商店區優先套用工業地板、商店招牌、交易告示與玻璃屋商盟色塊。
- 守衛巡邏區優先套用鹽脊守望徽章、直線護板與警告 decal。
- 前哨外觀優先套用焦黑木材、粗布旗幟、倉庫標誌與方向牌。
- 野外建造區優先套用乾裂沙地、水源警告、荒野危險標記與工坊標誌。

## 驗收

- 風格一致但不單色化。
- 灰盒材質覆蓋率逐 Gate 提升。
- 生成圖都有 metadata 與 manifest 記錄。
- Prompt pack 與 manifest 互相覆蓋。
- UE 匯入後不能破壞既有移動、戰鬥、交易、建造、生產、存檔與讀檔流程。
