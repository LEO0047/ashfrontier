# Ashfrontier 派系徽章與旗幟 Prompt Pack

## 共通設定

用途：1024x1024 派系徽章、旗幟與 banner 來源圖。徽章需在地圖、角色面板與警告通知中清楚可讀；旗幟需像日曬粗布，可用於場景 banner 材質。三個派系不只換顏色，必須有不同形狀語言。

共通反向提示：不要外部遊戲專有名稱、商標、既有角色、既有派系、既有地點、可辨識品牌、浮水印、真實軍徽、真實國徽、真實國旗、現代公司 logo。

## 徽章 Prompts

### `af_v02_emblem_saltwardens`

鹽脊守望原創派系徽章，直線城門與矛尖輪廓，灰白鹽痕與暗鐵，簡潔可讀，偏軍事與秩序。

### `af_v02_emblem_glasshouse`

玻璃屋商盟原創派系徽章，圓弧秤盤與鉚釘貨牌輪廓，氧化銅與褪色赭黃，簡潔可讀，偏商貿與工業。

### `af_v02_emblem_dustrunners`

塵路行者原創派系徽章，斜線裂片、繩結與陶片輪廓，風沙褐與日曬紅布，簡潔可讀，偏荒野與游牧。
