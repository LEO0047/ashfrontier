# Ashfrontier Decal 與 Signage Prompt Pack

## 共通設定

用途：1024x1024 場景 decal、告示牌與 signage。文字只使用繁體中文短字，粗、少、遠距離可讀。材質需符合粗木、日曬布、氧化金屬、鹽蝕砂岩與砂塵磨損。若生成文字不可讀，後續可改用 UE 文字或手工覆蓋，但 metadata 必須記錄。

共通反向提示：不要外部遊戲專有名稱、商標、既有角色、既有派系、既有地點、可辨識品牌、浮水印、真實政府標誌、現代交通標誌、現代企業 logo、英文店名。

## Asset Prompts

- `af_v02_decal_shop_sign`：商店招牌，繁體中文字「補給」，粗木牌，氧化銅邊框，砂塵磨損，遠距離可讀。
- `af_v02_decal_restricted_warning`：禁區警告，繁體中文字「禁區」，直線護板圖形，灰白鹽痕與暗鐵。
- `af_v02_decal_guard_notice`：守衛告示，繁體中文字「停步」，矛尖與城門符號，日曬布底，低飽和藍灰。
- `af_v02_decal_broken_direction`：破損方向牌，繁體中文字「北門」，裂片木牌，繩結，砂塵與焦黑邊緣。
- `af_v02_decal_workshop_mark`：工坊標誌，繁體中文字「工坊」，鉚釘貨牌與工具剪影，氧化金屬。
- `af_v02_decal_warehouse_mark`：倉庫標誌，繁體中文字「倉庫」，粗布貨牌，箱形輪廓，砂塵磨損。
- `af_v02_decal_water_warning`：水源警告，繁體中文字「鹹水」，水壺剪影與鹽痕，日曬布底。
- `af_v02_decal_trade_notice`：交易告示，繁體中文字「交易」，秤盤符號，氧化銅邊框，粗布底。
- `af_v02_decal_gate_mark`：城門標記，繁體中文字「城門」，階梯式護板與矛尖，鹽蝕砂岩底。
- `af_v02_decal_wild_danger`：荒野危險標記，繁體中文字「危險」，斜線裂片與陶片符號，風沙褐與焦黑邊緣。
