# Ashfrontier 生成美術風格聖經

## 核心方向

Ashfrontier v0.2 的生成美術要呈現荒漠邊境、鹽風侵蝕、破敗工業、半崩壞殖民城鎮與低科技高磨損。畫面應乾燥、粗糙、日曬、修補痕跡明顯，避免乾淨、華麗、科幻感過強或可辨識外部 IP 風格。

## 共通正向描述

- 乾燥荒漠邊境，低飽和砂土色，鹽蝕白痕，氧化金屬，粗布，焦黑木材。
- 手工修補、鉚釘、繩結、拼裝板、舊布旗、砂岩牆、陶片與骨質感裝飾。
- PBR-friendly surface, readable silhouette, game asset concept, no watermark.
- 材質需要可用於 UE5 / Metal 相容 prototype，避免過重細節與不可讀雜訊。

## 共通反向描述

不要外部遊戲專有名稱、商標、既有角色、既有派系、既有地點、可辨識品牌、浮水印、攝影棚文字、現代商標、過度乾淨表面、霓虹科幻、華麗宮廷、血腥獵奇、真人名人肖像。

## 色彩與明度

- 主色：砂土、灰白鹽痕、氧化鐵、褪色布、焦黑木、暗銅。
- 輔色：低飽和藍灰、風化赭黃、日曬紅布。
- 對比：icon 與徽章可提高輪廓對比；大面積場景貼圖保持低飽和。

## 三個派系的形狀語言

- 鹽脊守望：直線、城門、矛尖、階梯式護板，偏秩序與防衛。
- 玻璃屋商盟：圓弧、秤盤、鉚釘、貨牌，偏商貿與工業。
- 塵路行者：斜線、裂片、繩結、陶片，偏荒野與游牧。

## 生成與挑選規則

- 每張入選圖片都必須有 metadata。
- 入選理由需說明可讀性、用途與是否符合世界觀。
- UI icon 必須在 64x64 預覽仍可辨識。
- 表面材質優先選擇可裁切、可重複使用、細節均勻的圖。
- Signage 可以包含繁體中文短字，但必須避免過小與難辨識。
