# Ashfrontier UI Icon Prompt Pack

## 共通設定

用途：512x512 物品 UI icon。背景簡單，剪影清楚，64x64 仍可辨識。材質要符合荒漠邊境、鹽風侵蝕、低科技高磨損。Icon 不使用真實品牌、現代符號或外部 IP 標誌。

共通反向提示：不要外部遊戲專有名稱、商標、既有角色、既有派系、既有地點、可辨識品牌、浮水印、現代包裝、真實國旗、真實醫療標誌、霓虹發光戰利品。
