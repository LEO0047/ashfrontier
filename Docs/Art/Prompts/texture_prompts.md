# Ashfrontier 表面貼圖 Prompt Pack

## 共通設定

用途：PBR-friendly 表面來源圖，後續會處理成 basecolor、roughness approximation 與 normal approximation。優先 1024x1024 或 2048x2048，power-of-two，構圖需可裁切、可近似重複使用，不要文字、物件、浮水印或明顯方向光影。

共通反向提示：不要外部遊戲專有名稱、商標、既有角色、既有派系、既有地點、可辨識品牌、浮水印、現代企業標誌、過度乾淨表面、霓虹科幻、華麗宮廷、真人照片。

## Asset Prompts

### `af_v02_surface_cracked_sand`

可近似無縫重複的乾裂沙地 PBR base texture，低飽和砂土色，細小鹽晶，風蝕裂縫，沒有文字與物件，適合 UE 地表材質。

### `af_v02_surface_salt_stone`

可近似無縫重複的鹽蝕砂岩 PBR base texture，灰白鹽痕，砂岩孔洞，粗糙石面，低飽和，適合城牆與岩壁。

### `af_v02_surface_rusted_metal`

可近似無縫重複的生鏽金屬 PBR base texture，暗鐵板，鉚釘痕，氧化橘褐，刮痕與修補痕跡，適合低科技工業外殼。

### `af_v02_surface_sun_bleached_cloth`

可近似無縫重複的日曬舊布 PBR base texture，粗織纖維，褪色布料，砂塵污痕，邊緣磨損感，適合旗幟、棚布與 UI 背板。

### `af_v02_surface_blackened_wood`

可近似無縫重複的焦黑木材 PBR base texture，炭化邊緣，裂紋木板，灰塵與刮痕，適合棚架、防線與回收木板建築。

### `af_v02_surface_bone_ceramic`

可近似無縫重複的骨質陶片 PBR base texture，灰白陶片，粗糙孔洞，繩結刮痕，手工拼接感，適合荒野飾件與標記。

### `af_v02_surface_industrial_floor`

可近似無縫重複的粗糙工業地板 PBR base texture，拼裝鐵板，砂土卡縫，舊螺栓，磨損通道，適合商店區與工坊地面。

### `af_v02_surface_weathered_wall`

可近似無縫重複的風化城牆 PBR base texture，砂岩灰泥，氧化金屬補片，鹽霜垂痕，裂縫修補，適合主城與前哨牆體。
