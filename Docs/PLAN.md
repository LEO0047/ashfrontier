# Ashfrontier 開發計畫

Ashfrontier 是一個瀏覽器可執行的 3D 單機小隊沙盒 RPG 原型。第一版以 TypeScript、Vite、Three.js 與資料驅動系統完成可玩的荒原小隊 loop。

## 階段

1. 階段 0：專案啟動、repo 骨架、文件與 CI。
2. 階段 1：3D 場景、攝影機與可操作角色。
3. 階段 2：小隊選取、命令系統與路徑移動。
4. 階段 3：NPC、巡邏、城鎮與資料驅動內容。
5. 階段 4：即時戰鬥、部位傷害、倒地與搬運。
6. 階段 5：招募、交易、庫存與物品系統。
7. 階段 6：基地建設、資源採集與生產鏈。
8. 階段 7：派系關係、犯罪、禁區與守衛反應。
9. 階段 8：存讀檔、整合打磨、效能報告與最終可玩版。

## 交付原則

- 每個階段都產出可執行增量。
- 核心邏輯與 Three.js renderer 解耦。
- 內容使用原創命名，不使用受保護或高相似度名稱。
- 驗證先於 commit，push 目標為 `origin/main`。
